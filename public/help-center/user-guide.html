Pralux HR-App

Workforce intelligence platform

User Guide

This guide explains how to use the full Pralux HR-App workspace, including login, dashboards, employee management, attendance workflows, leave operations, reimbursement, reports, activity logs, and Help Center access.

Who This Guide Is For

  • HR and Admin users managing employee records and operational workflows.
  • Managers reviewing attendance and approval-related tasks.
  • Employees using attendance, leave, reimbursement, and profile self-service flows.

What Is Covered

  • Login and role-based access
  • Dashboard usage by role
  • Employee List and Add Employee flow
  • All Employee Attendance subflows
  • Reports, reimbursement, activity logs, and Help Center

1. Sign In

Employee Login

  1. Open the login page.
  2. Enter the employee username.
  3. Enter the password provided by HR.
  4. Click Sign In.

Demo Accounts

  1. Use the demo account panel on the right side.
  2. Select the target role.
  3. Click the preferred demo account row.
  4. The app will create the session automatically.

2. Dashboard

HR And Admin Dashboard

  • Review workforce totals and attendance status.
  • Track latest employees and department insights.
  • Monitor activity and reporting widgets.

Employee And Manager Dashboard

  • Review personal attendance summary.
  • Monitor recent records and requests.
  • Use the quick attendance action.

3. Employee List

This module is used by HR and Admin users as the master workspace for employee records.

Main Actions

  • Add employee
  • Edit employee data
  • Manage department and position references
  • Upload documents
  • Reset employee passwords
  • Configure app login access

Add Employee Step By Step

  1. Open Employee List.
  2. Click the add employee action.
  3. Fill all required tabs before saving.
  4. Upload documents if available.
  5. Save the employee and confirm the record appears in the table.

Personal Info Tab

  • NIK
  • Full name
  • Email
  • Birth place and birth date
  • Gender
  • Marital status and marriage date when applicable
  • Address
  • Identity card number
  • Phone number

Education Tab

  • Education summary
  • Education history rows
  • Level, institution, major, start year, and end year

Job Details Tab

  • Department
  • Position
  • Role
  • Status
  • Work location
  • Work type
  • Manager name
  • Contract type and contract period

Financial Details Tab

  • Position salary
  • Earnings and deduction components
  • Tax profile
  • Bank name
  • Bank account
  • App login enabled
  • Login username
  • Login password

Documents Tab

  • Select document type
  • Enter document title
  • Add notes if needed
  • Upload file
  • Preview or download uploaded file later

4. Employee Attendance

The attendance hub is the entry point for all attendance-related workflows. Each subflow has a dedicated page.

Attendance Hub

  • On Duty Request
  • Sick Submission
  • Leave Request
  • Half Day Leave
  • Submit Overtime
  • Leave Balance
  • Attendance Report for HR
  • Leave Report for HR

Clock In And Clock Out

  1. Use the quick action from the sidebar or attendance page.
  2. Confirm the attendance modal.
  3. Allow location or selfie access if required.
  4. Submit check-in or check-out.
  5. Confirm the status changes after submission.

On Duty Request

  1. Open On Duty Request.
  2. Set the request date.
  3. Enter the field work or assignment description.
  4. Submit the request.
  5. Review submitted requests and generated attendance records.

Sick Submission

  1. Open Sick Submission.
  2. Set start and end date.
  3. Enter the sick leave reason.
  4. Upload the doctor letter or medical evidence if available.
  5. Submit and monitor the status.

Leave Request

  1. Open Leave Request.
  2. Select one of the leave types allocated to the employee.
  3. Choose start and end date.
  4. Enter the reason.
  5. Upload a supporting document when needed.
  6. Submit and review the leave table.

Half Day Leave

  1. Open Half Day Leave.
  2. Choose the date.
  3. Select Morning or Afternoon.
  4. Enter the reason.
  5. Submit and review the balance impact.

Submit Overtime

  1. Open Submit Overtime.
  2. Choose the overtime date.
  3. Enter the duration in minutes.
  4. Add the overtime reason.
  5. Submit and review overtime records.

Leave Balance

Use this page to review leave allocations, carry-over, available balance, and sick leave usage.

Attendance Report

HR users can monitor attendance coverage and team-wide attendance data from this page.

Leave Report

HR users can review leave history and open supporting documents from the leave report page.

5. Leave System

The Leave System module supports HR and Admin visibility for leave-related administration.

  • Review leave administration screens
  • Review balance or leave flow context
  • Use the module alongside the attendance-based leave workflows

6. Reimbursement

Employee Flow

  1. Open Reimbursement.
  2. Select the claim type.
  3. Enter amount, receipt date, and remarks.
  4. Upload the receipt or supporting file.
  5. Submit the request.

Review Flow

  • Review reimbursement queue
  • Check approval status
  • Open supporting file references
  • Process approvals where allowed

7. Reports

  1. Open Reports.
  2. Choose the target report.
  3. Apply filters or period selection.
  4. Review preview data.
  5. Export when needed.

8. Activity Logs

  • Track who changed what
  • Filter by module, role, target, and date
  • Review event summary, actor, and affected record

9. Profile

  • Review personal profile data
  • Review department and position
  • Review work-related details

10. Help Center

  • Open the Help Center from the sidebar utility section.
  • Read the online guide.
  • Download the PDF version.
  • Use screenshots to match each module.
